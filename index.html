Estrutura Básica do HTML

(voltar)

O HTML é uma linguagem de marcação, sua função é definir a estrutura de uma página web. Sua estrutura consiste em certos elementos, onde cada um tem funções e propriedades, veja mais na imagem abaixo:

O "<!DOCTYPE html>" define o tipo de documento, e sua versão, o nome "DOCTYPE" é uma instrução, para que os navegadores ultilizem a versão mais atual do html.

Já o "<head>" é responsável por conter os metadados da página, como títulos, scripts, bibliotecas e outros elementos de configuração.

O "<title>" define o título da página, que é exibido na aba dos navegadores.

Por fim, o "<body>" é a tag responsável por mostrar todo contéudo ao usuário, é o corpo da página que contém todas as informações visíveis.

Tags Principais e suas Funções

(voltar)

As tags HTML são blocos de construção, que são de extrema importância nas páginas webs. Há uma variedade de tags, algumas cairam em desuso com o tempo mas outras se tornaram comuns no desenvolvimento web, como por exemplo:

  • "<h1>" a "<h6>": Definem títulos de diferentes níveis de importância, quanto maior o numéro menor a importância.
  • "<p>": Define um parágrafo de texto.
  • "<a>": Define um hiperlink.
  • "<img>": Insere uma imagem na página.
  • "<ul>" a "<ol>": Define não ordenadas e ordenadas , respectivamente.
  • "<li>": Define um item de lista.
  • "<input>": Define um campo de entrada, que pode ser aceitar tipos numéricos ou texto, além de poder ser um botão.

Atributos das Tags

(voltar)

As tags HTML podem conter atributos que podem ser usados no CSS e JS fornecem informações adicionais sobre os elementos. Os atributos mais comuns são:

  • "id": Identificador único para um elemento, não pode ser re-usado.
  • "class": Permite aplicar estilos CSS em multiplos elementos.
  • "src": Especifica o local de origem de um recurso, podendo ser imagens, scripts ou bibliotecas.
  • "href": Define o destino de um hiperlink
  • "alt": Fornece um texto alternatino para elementos como imagens (importante para acessibilidade).

Formatação de Texto com HTML

(voltar)

O HTML possui diversas tags de formatação de texto, que podem melhorar a legibilidade e a aparência das páginas web. Dentre elas temos:

  • "<strong>": Ultilizado para colocar negrito no texto.
  • "<em>": Deixa o texto em itálico.
  • "<u>": Sublinha o texto.
  • "<blockquote>": Indica uma citação longa.
